# Online Dress Buying Website – Single Page

This is my first web development project, built using HTML and CSS.
It is a simple and responsive online dress shopping website designed as a single-page layout with smooth scrolling navigation.

## Project Description

The Online Dress Buying Website is a basic e-commerce style webpage created to display different dress collections in a clean and attractive layout.

This project helped me understand the fundamentals of HTML structure, CSS styling, layout design, and smooth scroll behavior. It does not use JavaScript or backend technologies — it is purely frontend based.

## Features
- Single Page Layout
- Dress Collection Section
- Simple and Clean UI
- Built using only HTML & CSS
  
## Technologies Used
- HTML5
- CSS3
